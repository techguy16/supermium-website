    Supermium FAQ

    If you have any questions about Supermium, you may find an answer here.
    Got a question that isn't answered here? Go to this page.

    Last updated: September 8, 2024

    What is Supermium?

    Supermium is a Chromium-based web browser. It is frequently updated, and developed to ensure compatibility with legacy operating systems.
    You can learn more about Supermium here.

    Where can I get it?

    You can get Supermium here from the website, or you can download it from GitHub.

    Who is Supermium developed by?

    Supermium is developed by win32. Learn more about him and his other projects.

    Why does Supermium get flagged by my antivirus software?

    Your antivirus software and/or browser may pick up the Supermium installer as malware. This is a false positive.
    Supermium does not contain any malware, provided that you've installed it from the official website or GitHub.

    How come there are no Mac OS and Linux releases?

    At the moment, only Windows releases are made. Supermium was originally developed to have an up to date browser for older Windows operating systems.
    The project has evolved since to being more than an updated browser for older Windows operating systems, however, currently Windows is still the priority release target and user audience.
    Builds for Mac and Linux systems may be explored, but at this time there are no plans.

    Why is Supermium usually a few versions behind from the latest Chromium release?

    Rebasing Supermium to a newer version is a complex and time consuming process that requires many manual changes.
    The browser is rebased to a newer version of Chromium every so often. This allows for a browser that is fairly up to date and secure to use.
    In terms of security, being a few versions behind from the upstream release does not necessarily present any security risks.
    Security patches for any major vulnerabilities are applied to Supermium and released as soon as possible, even if Supermium is not on the latest version.

    Why do some streaming services (Netflix, Spotify, etc) not work with Supermium?

    Many streaming services use a proprietary DRM by Google known as Widevine. Most Chromium forks like Supermium do not have signed Widevine.
    As such some streaming services may give you an error when loading content. Supporting Widevine in Supermium would unfortunately be costly.

    Does Supermium automatically update when there is a new release?

    Currently Supermium does not automatically update. However, you can be notified of any new releases from the RSS feed, which you can add to your RSS reader.

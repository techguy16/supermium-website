    Obtaining extensions for Supermium

    Supermium is fully compatible with Google Chrome extensions. For this reason, you may be thinking, "What is the point of this page? Everyone knows where to download Chrome extensions!".
    While it is true that the Supermium browser can install Chrome extensions, it is important to note that Google is currently undergoing changes to their extensions manifest. In effect, some extensions (such as uBlock Origin) will soon become limited in functionality.

    The Supermium project is fully commited to ensuring that these changes do not impact the Supermium browser.
    As such, alternative sources to obtain extensions impacted by the new manifest are provided below. The list will be updated as the situation develops.

    Note that you may need to enable Developer Mode to install extensions from outside of the Chrome Web Store.

    Sources

    Chrome Web Store (official)
    The official Google Chrome Web Store

    Crx4Chrome (unofficial)
    Features archived versions of extensions

    Recommended Extensions

    enhanced-h264ify
    Use fewer resources during YouTube playback by using H.264 instead of VP8/VP9

    uBlock Origin
    An efficient content blocker. Direct link to the official Git with all releases in unpacked form
